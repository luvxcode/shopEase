# 🛒 ShopEase - E-Commerce Website


## 📌 About Project

ShopEase is a responsive e-commerce website developed using HTML, CSS, and JavaScript.

The project provides a simple online shopping experience with product listing, cart management, and contact features.



## 🚀 Features

✅ Responsive Design  
✅ Product Display  
✅ Add to Cart Functionality  
✅ LocalStorage Cart System  
✅ Cart Quantity Update  
✅ Remove Products from Cart  
✅ About Page  
✅ Contact Form Validation  
✅ 404 Error Page  


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
